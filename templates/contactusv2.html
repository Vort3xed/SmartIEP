Track student progression faster than ever

SmartIEP allows you to upload your student's IEPs and track their grade progression over time. Quickly switch between multiple different students and quickly download an automatically generated student report. Read the features we have below.

Secure Sign-in

Enter the registered email and password into the corresponding fields. To view the entered password, a "Show Password" button is provided. Upon entering the email and password, press the "Submit" button to log in. Incorrect login credentials will request for re-entry.

Student Creation

Enter the name, MCPS ID, and Grade of the student to be added. Next, select the Date of Birth drop down and click on the correct date. Enter the disability and select the case manager for the student from the dropdown menu. Enter the date of last annual review and click "Submit". The student is now added to the database and can be viewed on the "Students" page.

Edit Student Information

Select the pencil icon to open the editing menu. The menu to edit the fields will appear below.

Goals and Objectives

To add a goal, enter the goal's name into the text box and select the subject category. Click the green "+" button to add it. To add an objective, repeat the above steps and the objective will add as a sub-bullet under the goal. To remove a goal or objective, select the red trashcan icon. To change the progress indicator between: Not started, In progress, Completed, select the bar graph icon. The text color will change to represent these changes.

Viewing Student Logs and Tasks

Select either the Logs or Task button. You will be redirected to a new page containing an organized list of the appropriate information. The logs page will generate a graph based on quantitative metrics such as test scores. New logs can be created by entering the date, notes, future actions if necessary, and the assignment score if needed. Select the "+" button to save and create the new log. The graph will update and represent these scores by percent accuracy and date of creation.
